BAVICU SE VISE SA if U NAREDNOJ SERIJI BRANCH-EVA ALI EVO DA PROBAM DA NAPISEM PAR IF STATEMENT-A

```py
>>> if "something":
...     print("Nick")
...

```

```py
>>> my_dict = {8: "Nick"}
>>> my_dict_2 = my_dict
>>> if my_dict is my_dict_2:
...     print("Idntity confirmed")
...

```
