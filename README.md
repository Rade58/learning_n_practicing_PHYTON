# COMPARING ZERO (`0`) WITH `False`

KADA COMPARE-UJES `0` SA `False`, KORISCENJEM `==` DOBICES `True`

```py
>>> 0 == False
True
```

## MEDJUTIM TO NE VAZI ZA DRUGE FALSY VREDNOSTI

```py
>>> "" == False
False
>>> [] == False
False
```
