# JA SAM RANJE VEC NA JEDAN NACIN PRAVIO LIST

EVO NEKOLIKO LISTA

```py
>>> []
[]
>>> [1,2,3,4]
[1, 2, 3, 4]
```

# TAKODJE MOGUCE JE KORISTITI METODU KOJA PRAVI LIST

```py
>>> list()
[]
>>> list([8,6,4,8])
[8, 6, 4, 8]
>>> list("nesto")
['n', 'e', 's', 't', 'o']
>>> list("12345678")
['1', '2', '3', '4', '5', '6', '7', '8']
```
