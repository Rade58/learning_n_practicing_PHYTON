FOR LOOPS U PYTHONU TREBALE BI DA BUDU A LOT NICER NEGO ONE U JAVASCRIPT-U

```py
>>> colors = ["tomato", "olive", "crimson", "blanchedalmond"]
# POPRILICNO INTUITIVNO
>>> for color in colors:
...     print(color)
... 
tomato
olive
crimson
blanchedalmond
```
